# Study Organizer - Organizador Académico Inteligente 

## Descripción del Proyecto

**Study Organizer** es una aplicación móvil diseñada para ayudar a estudiantes a organizar su vida académica de manera eficiente. La aplicación centraliza la gestión de tareas, horarios de estudio y materiales de aprendizaje en una plataforma intuitiva, mejorando la productividad y el rendimiento académico.

### Características Principales
- **Gestión Completa de Tareas**: Crear, editar, eliminar y organizar asignaciones académicas
- **Administración de Materias**: Registrar cursos con horarios, profesores y aulas
- **Persistencia de Datos Local**: Almacenamiento con SQLite para datos persistentes
- **Interfaz Intuitiva**: Diseño Material Design con navegación simple
- **Priorización Inteligente**: Sistema de prioridades (Alta, Media, Baja) con colores diferenciados

## Instalación y Ejecución

### Prerrequisitos
- Flutter SDK 3.0 o superior
- Android Studio o VS Code
- Android SDK 21+ (para emulador o dispositivo físico)
- Git instalado

### Paso 1: Clonar el Repositorio
- https://github.com/fatimafgz/Proyecto-Final
- cd proyecto_final
### Paso 2: Obtener dependencias
- flutter pub get
### Paso 3: Ejecutar la aplicación
- flutter run
